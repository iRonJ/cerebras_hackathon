3D UMAP visualization of JPEG/HEIF/PNG images from Photos Library using CLIP embeddings. Interactive 3D scatter plot with image preview on click, distance to nearest neighbor coloring, and size-based marker scaling. Include folder path input and browse button. Use 'directory' parameter for API calls. Add macOS file commands (open and reveal in Finder) to the image preview info box.

Projection mapper tool with OpenCV-style remapping and warping. Create 4-sided polygons, drag vertices to reshape, upload media files and apply perspective transformation and other transformations to warp media so corners match all polygone corners. Features: toggle between fit-inside and stretch-to-corners modes, save/load projects, responsive design with sidebar controls and canvas workspace.
